undo.UndoManager

CKEDITOR.plugins.undo.UndoManager

重做/撤消功能的主要逻辑。

属性

limit : Number     只读

堆栈中的最大快照数量。 通过CKEDITOR.config.undoStackSize进行配置。

locked : Object     只读

当锁定属性不为空时，撤销管理器被锁定，因此禁止保存或更新等操作。

管理员可以分别通过锁定和解锁方法锁定和解锁。

默认为null

previousKeyGroup : Number     只读

包含基于keyGroups的先前处理的密钥组。 -1 表示一个未知组。

默认为 -1

strokesLimit : Number     只读

在一个撤消步骤中键入/删除的最大字符数。

默认为 25

strokesRecorded : Array     只读

存储按键次数的数组，连续计数。 使用keyGroups成员作为索引。

注意：按键计数将在达到每个快照的字符数限制后重置。

默认为 [0，0]
