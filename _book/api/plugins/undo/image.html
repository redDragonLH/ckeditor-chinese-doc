undo.image

CKEDITOR.plugins.undo.Image

包含编辑器内容和特定时间点的快照。

属性

bookmarks : Object[]     只读

表示图像中的选择的书签。bookmark2 对象的数组，请参阅CKEDITOR.dom.range.createBookmark2以进行定义。

contents : String     只读

编辑内容。

方法

constructor( editor, [ contentsOnly ] ) → Image

创建一个Image类实例。

参数

    editor :editor

        在其上创建图像的编辑器实例。

     [contentsOnly]：Boolean

        如果设置为true，图像将只包含没有选择的内容。

返回

    Image类

equalsContent( otherImage ) → Boolean

  Image.equalsContent( otherImage )

参数

     otherImage : Image

         要比较的 Image

返回

     Boolean

         如果otherImage中的内容相同，则返回true。

equalsSelection( otherImage ) → Boolean

参数

     otherImage : Image

         要比较的 Image

返回

     Boolean          如果otherImage中的选择相同，则返回true。
